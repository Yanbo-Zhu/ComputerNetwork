# 2 合并数据包
Hauptmenu -> file ->merge

可以选择两个数据包 合并其, 然后通过 wirshark 打开 

# 3 打印数据包

Hauptmenu -> file ->print 

## 3.1 导出数据包 
Hauptmenu -> file -> export specified Packets

1. 只导出标记的数据包
2. 只爆出选择的数据包
3. 导出为csv格式
